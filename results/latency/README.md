# Latency and Throughput Results

Files:

- `guardrail_latency_throughput.csv`: tabular latency and throughput measurements.
- `guardrail_latency_throughput.json`: structured latency and throughput measurements.
- `run_speed_bench.sh`: command scaffold used for vLLM speed benchmarking.

Metrics:

- `samples_per_second`: higher is better.
- `p50_ms`: median per-sample latency in milliseconds.
- `p95_ms`: 95th percentile per-sample latency in milliseconds.
